# Чеклист — gbc-analytics-dashboard

```
- [ ] Прочитать README.md — все 5 шагов и критерии сдачи.
- [ ] .env.example обновлён; в git нет .env с секретами.
- [ ] Заказы из RetailCRM попадают в Supabase (скрипт или job).
- [ ] Дашборд читает данные из Supabase и строит график.
- [ ] Уведомление в Telegram при заказе > 50 000 ₸ (проверить расчёт суммы).
- [ ] README: промпты AI, где застряли, как решили + ссылки/инструкции деплоя.
```

## Подсказки

- Сумма заказа для порога 50 000 ₸: агрегировать `items` (`initialPrice` × `quantity`) из ответа RetailCRM или из своей схемы в Supabase.
- Демо RetailCRM: использовать URL и ключ API из личного кабинета демо-аккаунта.
- Перед объявлением «готово» прогнать документированные команды сборки/скрипта.
