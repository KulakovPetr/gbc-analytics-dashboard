# Локальные папки `.cursor/` и `local/`

Они **не хранятся в этом репозитории** (в `.gitignore`), чтобы не попадали на GitHub и в деплой Vercel.

- **`local/`** — создайте у себя вручную: `mkdir local` и складывайте черновики, промпты, скриншоты до отбора. В git не коммитится.
- **`.cursor/`** — правила и скиллы Cursor держите **локально** или в [глобальных настройках Cursor](https://cursor.com/docs); в общий репозиторий задания не включаются.
