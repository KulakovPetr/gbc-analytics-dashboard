# Локальные материалы (не для сдачи)

Этот каталог предназначен для файлов, которые **не нужно включать в результат** тестового задания:

- черновики промптов для Claude / Cursor;
- экспорт чатов, заметки «где застрял»;
- личные копии скиллов или правил;
- временные JSON, скриншоты до отбора;
- дубликаты `.env` под другим именем.

**Правило:** не кладите сюда единственный экземпляр важного кода — воспроизводимый код и скрипты сдачи лежат в корне репозитория (например `scripts/`, `app/`).

Содержимое `local/*` (кроме этого `README.md`) **игнорируется Git** — в GitHub не попадёт.
